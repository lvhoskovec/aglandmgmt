This repository contains code to visualize results from the Agricultural land management activities for conterminous US from 1980-2023 published on Dryad (DOI: 10.5061/dryad.t1g1jwtgm). 
The data are in 9 netcdf files. 
The code is written in R to read in the files, access metadata, and visualize the dataset using ggplot2. 
Metadata are described in the .nc files and on the Dryad ReadMe file.
